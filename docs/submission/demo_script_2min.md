# 2-Minute Demo Voice-Over Script

Target length: 1:40 to 2:00.

## 0:00 - 0:15 — Intro

"This is Meeting Notes Organizer. It helps students and teams turn messy meeting notes into structured summary, decisions, and action items."

## 0:15 - 0:35 — Auth and user isolation

- Open login page
- Register or login
- Mention that each user has private history

Voice-over:
"Version 2 includes authentication. Each user's meeting history is isolated and protected."

## 0:35 - 1:05 — Core analysis flow (V1)

- Open Analyze page
- Paste raw meeting notes
- Click Analyze
- Show generated summary, decisions, action items

Voice-over:
"The core feature from Version 1 is the end-to-end analysis pipeline using a free Qwen API through the backend."

## 1:05 - 1:25 — Version 2 features

- Open History page
- Show saved records and pagination
- Open one result and export Markdown

Voice-over:
"Version 2 adds history, better UX, and export to Markdown for quick sharing."

## 1:25 - 1:45 — Deployment

- Show running app via VM IP in browser
- Optional: show `docker compose ps`

Voice-over:
"The app is fully dockerized: frontend, backend, and PostgreSQL, deployed on Ubuntu VM with Docker Compose."

## 1:45 - 2:00 — Wrap-up

"So this project goes from idea to deployed product with a useful core workflow and practical improvements based on TA feedback."
