# Task 5 Links Template

Use these placeholders in Slide 5 and replace before submission.

- GitHub repository: `https://github.com/<username>/se-toolkit-hackathon`
- Deployed product: `http://<VM_IP>:5173` (or your custom domain)

## QR Code Tips

You need 2 QR codes:

- QR for GitHub repository link
- QR for deployed product link

Fast options:

1. Use any online QR generator and export PNG.
2. Insert PNGs on Slide 5 near each link.
3. Scan both QR codes from phone before final submission.
